## 💡 Sobre

O **Lumora** é um app que oferece funcionalidades de apoio e segurança para mulheres em situações de risco.  

🔒 Botão de emergência  
🎭 Modo disfarçado  
📍 Compartilhamento de localização  
👭 Comunidades protegidas
